### admin 端
- vue2-editor 富文本编辑器


### server 端
- cors 跨域
- mongoose 数据库连接
- inflection 单复数转换
- multer 文件上传
- bcrypt 加密
- jsonwebtoken token
- http-assert 验证抛出错误


### web 端
- vue-awesome-swiper 幻灯片
- www.spritecow.com 精灵图定位辅助网站
